**💫 Personalized DeFi Activities**

## 🚀 Getting Started

### Quick Start Guide

```bash
# Clone the ecosystem
git clone link-to-ocnetwork-repo
cd OCNetwork

# Install all dependencies
npm run install:all

# Configure your environment
cp frontweb/.env.example frontweb/.env
cp agentOC/.env.example agentOC/.env

# Deploy the smart contracts
cd contracts
sui client publish --gas-budget 1000000000

# Start the ecosystem
npm run start:all
```

### Environment Setup

**Frontend (.env):**
```env
VITE_SUI_NETWORK=testnet
VITE_PACKAGE_ID=0xyour-package-id
VITE_MARKETPLACE_ID=0xyour-marketplace-id
VITE_WALRUS_ENDPOINT=your-walrus-endpoint
```

**Agent Engine (.env):**
```env
AGENT_PRIVATE_KEY=your-secure-private-key
CONTRACT_PACKAGE_ID=0xyour-contract-package
ENABLE_AUTO_YIELD_FARMER=true
MAX_TRADE_AMOUNT_SUI=1000
MIN_PROFIT_THRESHOLD=0.1
```
