## 🛠️ 环境准备清单

### 必装工具
```bash
# 1. 安装 Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 2. 安装 Sui CLI
cargo install --locked --git https://github.com/MystenLabs/sui.git --branch testnet sui

# 3. 配置 Sui 网络
sui client

# 4. 安装项目依赖
cd ocnetwork
npm run install:all
```

### 推荐工具
- **代码编辑器**：VS Code + rust-analyzer + Sui 扩展
- **浏览器插件**：Sui Wallet
- **测试工具**：Sui CLI 测试套件

## 📚 核心学习内容

### 🦀 Rust 基础（2-3周）
- **第1周**：基础语法、控制流、函数
- **第2周**：所有权、借用、生命周期
- **第3周**：结构体、枚举、错误处理

### ⚡ Move 语言（1-2周）
- **第4周**：Move 基础、资源模型、能力系统
- **第5周**：对象编程、事件系统、权限管理

### 🌊 Sui 生态（2-3周）
- **第6周**：Sui 框架、对象模型、代币处理
- **第7周**：测试、部署、网络交互
- **第8周**：项目架构分析、最佳实践

### 🚀 项目实战（2-3周）
- **第9-10周**：功能扩展开发
- **第11-12周**：优化、部署、维护
